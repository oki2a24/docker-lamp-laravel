# docker-lamp-laravel
Laravel 用の Docker Compose です。

## 前提
- Laravel のプロジェクトは作成されているものとします。
  - 例えば、プロジェクトディレクトリ名を、`sample` として説明します。

## 構成
次のようなディレクトリ構成で使用します。

```
.
└── app
    ├── docker-lamp-laravel
    └── sample
```

## 導入

```bash
cd app/
git clone https://github.com/oki2a24/docker-lamp-laravel.git
```
